## 🚀 Overview
**Kayana Source Finance** adalah aplikasi manajemen keuangan pribadi berbasis web yang dirancang untuk kecepatan, keamanan, dan kemudahan penggunaan. Aplikasi ini memungkinkan pengguna untuk memantau arus kas (*Income* & *Expense*) secara real-time dengan integrasi database awan.

Dikembangkan oleh **Azka Danish Kayana**, project ini mengusung konsep *Minimalist Glassmorphism UI* yang nyaman di mata dan responsif di berbagai perangkat.

## ✨ Fitur Unggulan
* **Google Authentication**: Login aman sekali klik menggunakan akun Google.
* **Smart Entry**: Input transaksi cepat dengan kategori yang bisa dikustomisasi secara dinamis.
* **Real-time Balance**: Kalkulasi saldo otomatis (*Total In, Total Out, Net Balance*).
* **Custom Category**: Fitur "Lainnya" yang memungkinkan input kategori manual jika tidak ada di daftar.
* **Nuclear Reset**: Fitur hapus seluruh basis data transaksi untuk memulai pembukuan dari nol.
* **Cloud Persistence**: Data tersimpan aman di Firebase Firestore, akses dari mana saja.

## 🛠️ Tech Stack
* **Frontend**: HTML5, Tailwind CSS (Styling), JavaScript (ES6+).
* **Backend as a Service (BaaS)**: Firebase Authentication & Firestore Database.
* **Deployment**: GitHub Pages.
